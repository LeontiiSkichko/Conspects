
# Параметры

| Параметр | Длинный параметр | Описание                                                                                                              |
| -------- | ---------------- | --------------------------------------------------------------------------------------------------------------------- |
| -a       | --all            | Список всех файлов директории.                                                                                        |
| -A       | --almost-all     | Короче тоже что -a, но дольше вводить.                                                                                |
| -d       | --directory      | Выводит информацию о самом каталоге если использовать вместе с -l выведет дольше информации о нем.                    |
| -F       | -classify        | Добавляет в конец каждого имени символ индекатор.                                                                     |
| -h       | --human-readable | при использовании длинного формата вывода отображает размеры файлов не в байтах, а в величинах с единицами измерения. |
| -l       |                  | Выводит результаты с использованием длинного формата.                                                                 |
| -r       | --reverse        | Выводит результаты в обратном порядке. Обычно команда ls выводит результаты в алфавитном порядке.                     |
| -S       |                  | Сортировать результаты по размеру (size).                                                                             |
| -t       |                  | Сортировать результаты по времени (time) последнего изменения.                                                        |


# Длинный формат
  

| Поле             | Назначение                                                                                                                                                                                                                                                                                                                                         |
| ---------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| -rw-r-r--        | Права доступа к файлу. Первый символ указывает тип файла. Например, символ d - каталоги. следующие три символа сообщают о правах доступа для владельца файла, следующие три - для членов группы, которой принадлежит файл, и последние три - для всех остальных. Более полный обзор прав доступа тут: <вставить ссылку на статью о правах доступа> |
| 1                | Число жестких ссылок на файл. Подробнее о ссылках рассказывается тут: <вставить ссылку на статью о правах доступа>                                                                                                                                                                                                                                 |
| root             | Имя пользователя владеющего файлом.                                                                                                                                                                                                                                                                                                                |
| root             | Имя группы, владеющей файлом.                                                                                                                                                                                                                                                                                                                      |
| 32059            | Размер файла в байтах.                                                                                                                                                                                                                                                                                                                             |
| 2017-04-03 11:05 | Дата и время последнего изменения файла.                                                                                                                                                                                                                                                                                                           |
| oo-cd-cover.odf  | Имя файла.                                                                                                                                                                                                                                                                                                                                         |
