- **Первая строка (top):**
    - `top - 09:02:34`: Текущее системное время.
    - `up 14:46`: Время, прошедшее с последней загрузки системы (uptime).
    - `1 user`: Количество пользователей, вошедших в систему.
    - `load average: 0.00, 0.00, 0.00`: Средняя нагрузка на систему (количество активных процессов) за последние 1, 5 и 15 минут.
- **Вторая строка (Tasks - Процессы):**
    - `total`: Общее количество процессов.
    - `running`: Процессы, выполняющиеся прямо сейчас.
    - `sleeping`: Процессы в режиме ожидания (спят).
    - `stopped`: Остановленные процессы.
    - `zombie`: Зомби-процессы (завершенные, но не удаленные из таблицы процессов).
- **Третья строка (%Cpu(s) - Процессор):**
    - `us` (user): Процент времени ЦП, затраченный на пользовательские процессы.
    - `sy` (system): Процент времени ЦП, затраченный на системные процессы (ядро).
    - `ni` (nice): Время, затраченное на выполнение процессов с приоритетом nice.
    - `id` (idle): Время простоя ЦП.
    - `wa` (iowait): Время ожидания завершения операций ввода-вывода (I/O).
    - `hi`, `si`: Обработка аппаратных и программных прерываний.
    - `st` (steal): Время, «украденное» гипервизором у виртуальной машины.
- **Четвертая/Пятая строка (MiB Mem/Swap - Память):**
    - `total`: Всего памяти.
    - `free`: Свободная память.
    - `used`: Используемая память.
    - `buff/cache`: Память, используемая для буферов и кэша.
    - `avail Mem`: Память, доступная для запуска новых приложений (более точный показатель, чем `free`
2. Список процессов (Таблица)

- **PID**: Уникальный идентификатор процесса.
- **USER**: Имя пользователя, запустившего процесс.
- **PR**: Приоритет процесса (Priority).
- **NI**: Значение nice (чем ниже, тем выше приоритет).
- **VIRT**: Виртуальная память, используемая процессом (весь код, данные, разделяемые библиотеки, а также страницы, которые были выгружены).
- **RES**: Resident Set Size, объем физической оперативной памяти, которую процесс использует в данный момент (в Кб).
- **SHR**: Объем разделяемой памяти.
- **S**: Статус процесса:
    - `R` (Running): Работает/в очереди.
    - `S` (Sleeping): Спит.
    - `D` (Uninterruptible sleep): Ожидает ввода-вывода.
    - `Z` (Zombie): Зомби.
    - `T` (Stopped): Остановлен.
- **%CPU**: Процент процессорного времени, используемого процессом.
- **%MEM**: Процент физической памяти, используемой процессом.
- **TIME+**: Общее время ЦП, затраченное процессом.
- **COMMAND**: Команда, запустившая процесс. 

3. Горячие клавиши (Управление в top)

- `q`: Выход из утилиты.
- `h`: Справка.
- `Shift + m`: Сортировка процессов по использованию памяти.
- `Shift + p`: Сортировка процессов по нагрузке на ЦП (по умолчанию).
- `Shift + t`: Сортировка по времени (TIME+).
- `u`: Показать процессы конкретного пользователя.
- `k`: Убить процесс (kill).
- `1`: Развернуть статистику по каждому ядру ЦП отдельно.
