
| Сигнал      | Номер | Описание                                   | Действие по умолчанию            |
| ----------- | ----- | ------------------------------------------ | -------------------------------- |
| **SIGHUP**  | 1     | Разрыв связи с терминалом (Hangup)         | Завершение                       |
| **SIGINT**  | 2     | Прерывание с клавиатуры (`Ctrl+C`)         | Завершение                       |
| **SIGKILL** | 3     | Выход с дампом памяти (`Ctrl+\`)           | Завершение + Core Dump           |
| **SIGUSR1** | 9     | Безусловное немедленное завершение         | Завершение (нельзя игнорировать) |
| **SIGUSR1** | 10    | Пользовательский сигнал №1                 | Завершение                       |
| **SIGSEGV** | 11    | Ошибка сегментации (доступ к чужой памяти) | Завершение + Core Dump           |
| **SIGUSR2** | 12    | Пользовательский сигнал №2                 | Завершение                       |
| **SIGPIPE** | 13    | Ошибка записи в пустой канал (Pipe)        | Завершение                       |
| **SIGTERM** | 15    | Вежливое завершение (сигнал по умолчанию)  | Завершение                       |
| **SIGCONT** | 18    | Продолжить выполнение процесса             | Игнорируется / Продолжение       |
| **SIGSTOP** | 19    | Принудительная остановка (пауза)           | Остановка (нельзя игнорировать)  |
